# Project Vision

## What GameFinder Is

GameFinder is a video game discovery platform that helps people find games matching their constraints: age suitability, difficulty, genre, platform, play style, multiplayer support, content intensity, and time commitment.

The repository serves a dual purpose:

1. **Product** — A realistic video game recommendation catalog built as a TypeScript monorepo.
2. **Testbed** — A repeatable public benchmark for evaluating AI-assisted development workflows.

## Target Users

- **Parents** — Need age-appropriate games with clear content notes and session length estimates.
- **Educators** — Need games suitable for classroom or group contexts with bounded play times.
- **Casual players** — Want quick recommendations by genre, platform, and difficulty.
- **Hobbyists** — Want finer-grained expertise and play-style filters.
- **Store/community staff** — Need reference data and moderation tools.
- **Moderators/admins** — Curate suggestions and maintain catalog quality.

## Core Use Cases

- Find games by age range, difficulty, genre, platform, and play style
- Submit community age and expertise suggestions with rationale
- Compare official ratings with community guidance
- Favorite games and track personal suggestions
- Moderate pending suggestions and new game submissions
- Administer catalog entries and user roles

## Why Video Games Fit the AI Workflow Testbed

Video games combine structured metadata with subjective community judgment — similar to real products where validation, moderation, and RBAC all matter. The domain is familiar, supports web + mobile clients, and can grow incrementally without enterprise compliance baggage.
